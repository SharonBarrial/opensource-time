# Rick and Morty Character Viewer
This project is a web application that allows users to view information about characters from the popular animated series "Rick and Morty". It utilizes the Rick and [Morty API](https://rickandmortyapi.com/api/character/) to fetch character data and displays their image, name, status, species, and location.
# Features
  * Character Display: The application consumes the Rick and Morty API to fetch information about the main characters of the series.
  * Displayed Data: For each character, their image, name, status (alive or deceased), species, and current location are displayed.
  * User-Friendly Interface: The user interface provides a friendly and intuitive experience for navigating and exploring character information.
## Usage
  1. Clone this repository to your local machine.
  2. Install dependencies using npm install.
  3. Start the application using ng serve.
  4. Open your web browser and navigate to http://localhost:4200/ to see the application in action.
## Technologies Used
  * Angular: The frontend of the application is developed using the Angular framework.
  * TypeScript: The primary programming language used in the application.
  * Angular Material: Angular Material is used to design and style the user interface.
  * Rick and Morty API: The application consumes data from the Rick and Morty API to obtain information about the characters.
